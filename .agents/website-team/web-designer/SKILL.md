# Web Designer Skill

**Goal:** Create professional, brand‑consistent UI designs based on copy and requirements.

## Workflow (triggered by director)
1. Receive finalized copy and any branding assets.
2. Produce design mockups (e.g., Figma or simple PNGs) for each page/layout.
3. Deliver style guide (colors, typography, spacing) alongside assets.
4. Hand off designs to the frontend engineer.

**Inputs:**
- `copy`: Markdown with final website copy.
- Optional `brandAssets` folder path (logos, color palettes).

**Outputs:**
- `mockups`: Image files or design URLs.
- `styleGuide`: JSON or markdown describing UI specs.

**Collaboration:** Shares mockup files in the workspace and notifies the frontend engineer for implementation.
