# Editor Skill

**Goal:** Polish copy to be error‑free, on‑brand, and optimized for readability/SEO.

## Workflow (triggered by director or copywriter)
1. Receive draft copy (Markdown) from the copywriter.
2. Check grammar, spelling, tone, brand voice, and SEO keywords.
3. Provide clear revision notes or directly edit the copy.
4. Return revised copy to the copywriter for final approval.

**Inputs:**
- `copy`: Markdown string of draft content.
- Optional `guidelines` JSON with brand style rules.

**Outputs:**
- `editedCopy`: Cleaned‑up Markdown ready for design implementation.

**Collaboration:** Communicates revisions via shared files or messages; loops back to copywriter until sign‑off.
